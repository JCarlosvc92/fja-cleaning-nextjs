
# FJA Cleaning (Next.js)

## Quick start

```bash
npm i
npm run dev
```

## Deploy
- Vercel: import this repo and set the project
- Update Formspree endpoint in components/ContactForm.jsx (replace YOUR_FORM_ID)

## Config
- i18n locales: en (default), es
- Colors in tailwind.config.js (primary, accent, gold)
- Phone/address in Footer.jsx and schema JSON-LD

## Notes
- Replace /public/logo.webp with your high-res logo if needed
- Add images to /public and use <img src="/your.jpg" />
